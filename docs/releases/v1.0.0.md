# Steam Vault App v1.0.0

建议填写方式：

- Tag version：`v1.0.0`
- Release title：`Steam Vault App v1.0.0`
- Target：`main`

---

Steam Vault App 的首个公开版本，当前主线已经覆盖 Steam Guard 令牌管理、协议登录、手机验证器绑定、交易确认、扫码登录批准、本地加密备份和 WebDAV 云备份等核心能力。

## 本版本亮点

- 主密码 + Argon2id + Android Keystore + AES-GCM 本地加密 Vault
- 支持从 maFile、JSON、otpauth URI 和手动 `shared_secret` 导入 Steam 令牌
- 支持离线生成 Steam Guard 动态码
- 支持 Steam 协议登录与手机验证器绑定
- 支持 confirmations 拉取、批准和拒绝
- 支持 Steam 扫码登录请求的读取、账户选择和批准/拒绝
- 支持 Steam Session 本地保存、修复和恢复
- 支持本地加密备份导出 / 恢复
- 支持 WebDAV 云备份上传 / 恢复
- 支持备份 `metadata_summary`，便于核对备份中是否包含令牌、Session 和确认所需材料

## 已包含的主要能力

### 本地安全

- 主密码创建、解锁、锁定
- 主密码修改与 Vault key 重新包装
- 生物识别快速解锁基础能力
- 本地安全设置持久化

### Steam 令牌

- 多种格式导入已有令牌
- 令牌列表与详情页
- 动态码与倒计时展示
- 撤销码、设备 ID、identity secret 等敏感信息保护展示

### Steam 登录与手机验证器

- 协议登录与 challenge 处理
- 新增手机验证器与 finalize 流程
- 绑定完成后的本地令牌与 Session 持久化
- 对已有手机验证器账号的兼容导入与恢复引导

### 确认与扫码批准

- confirmations 列表拉取
- 单条交易 / 市场确认批准与拒绝
- 会话失效时的自动修复与重试
- Steam 扫码登录请求读取
- 选择账户后批准或拒绝二维码登录

### 备份与恢复

- 本地加密备份导出 / 恢复
- WebDAV 云备份手动上传 / 恢复
- 自动触发上传、失败重试、多版本保留
- 备份中包含 Vault、Steam Session、Steam Guard 相关档案

## 当前已知边界

- 锁定后 / 退出后的自动云备份时机仍在继续收口
- 部分 Steam Session 恢复边界仍需更多真机回归
- 当前仓库默认不附带预编译 APK，需要本地自行构建

## 构建方式

```powershell
./gradlew.bat :app:testDebugUnitTest
./gradlew.bat :app:assembleDebug
```
